# NeuroLearn AI

AI-powered learning assistant built with React, FastAPI and Ollama.

## Features

- AI Roadmap Generation
- PDF Upload
- AI Notes Summarization
- AI Quiz Generation
- Quiz Scoring
- Local LLM Support using Ollama + Phi3

## Tech Stack

Frontend:
- React
- TailwindCSS
- Axios

Backend:
- FastAPI
- Python

AI:
- Ollama
- Phi3

## Run Locally

Backend:
uvicorn main:app --reload

Frontend:
npm install
npm run dev
